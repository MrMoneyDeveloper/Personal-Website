
# Portfolio Word Color Map
**Project:** Personal Website / Portfolio  
**Purpose:** A structured word-to-color guide for highlighted keywords, brand names, platforms, tools, and capability terms used across the live site pages.

---

## Notes
This is a **site-friendly semantic color map**, not a strict legal brand-guideline file.

Use these colors to:
- highlight key words inside headings, leads, project titles, bullets, tags, and feature lists
- keep recurring terms visually consistent across pages
- help users scan technical concepts more easily
- reinforce brand and platform recognition without making the layout noisy

Recommended rule:
- color only the **most important phrase** in a sentence
- avoid coloring more than **1 to 3 terms per block**
- keep long paragraphs mostly neutral
- use stronger glow only for hero headings, project cards, tags, and section titles

## Final Recommendation
If you want the cleanest result, build this as:
1. **brand words**
2. **tool/platform words**
3. **technical concept words**
4. **role/service words**

That way the color system stays readable and does not turn into rainbow text chaos.
