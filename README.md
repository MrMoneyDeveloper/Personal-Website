# Placeholder Studio — Full Front-End Redesign

This repository has been fully reworked into a placeholder-first website shell.

## Routes
- `/` Home
- `/about.html` About
- `/services.html` Services
- `/projects.html` Projects
- `/news.html` News
- `/contact.html` Contact
- Legacy aliases retained:
  - `/skills.html` → Services placeholder context
  - `/experience.html` → News placeholder context

## Design Goals Implemented
- Premium dark immersive visual language
- Consistent spacing/grid/card/button/typography system
- Motion-led section choreography and hover depth
- Vanta-inspired WebGL background treatment
- Responsive desktop/tablet/mobile behavior
- Placeholder-only copy throughout

## Tech
- Jekyll static pages/layouts
- Vanilla CSS design system (`assets/css/style.css`)
- GSAP + ScrollTrigger for reveal/parallax-like motion
- Vanta NET background for ambient movement

## Removed Legacy Surface
Old personal branding content, RPG-specific sections, timeline narratives, company subpages, and game-specific scripts/styles were removed and replaced by modular placeholder components.
